# Записка по реализуемости — карта памятников

> К итерации от 11.07.2026. Что уже сделано, что реализуемо и на каких условиях, какие решения нужны от заказчика.

## Сделано в этой итерации

1. **Поиск** — строка вверху по центру карты. Одна строка находит: памятники по названию, категории («исторические памятники»), округа («Калининский район»). Подсказки с автодополнением, выбор — мышью или стрелками+Enter.
2. **Фильтры пересобраны** — вместо панели слева теперь компактные чипы категорий под строкой поиска (в один тап, как в Яндекс.Картах). Блок статистики и его неиспользуемый код удалены из проекта.
3. **Карточка памятника расширена** (правая панель увеличена с 400 до 480 px):
   - галерея: несколько фото **и видео** вперемешку (свайп/стрелки);
   - **рейтинг** — звёзды, значение пока задаётся в админке (см. открытые вопросы);
   - **секции описания** («Архитектура», «История», …) — произвольный набор, добавляются в админке;
   - **аудиогид** — кнопка-плеер + текстовая версия («читать текстом»);
   - кнопка **«Пожертвовать на реставрацию»** — модальное окно; платёжная часть ждёт реквизитов.
4. **Мероприятия** — раздел в админке (ручной ввод: памятник, название, описание, даты). Памятники с мероприятием «на сегодня» пульсируют на карте, информация о событии показывается в карточке (сегодняшние выделены, ближайшие — списком).
5. **Редактор для контент-менеджера** дополнен: загрузка нескольких фото (было), видео (mp4/webm до 100 МБ), аудио аудиогида (mp3 и др. до 30 МБ), секции описания, рейтинг.
6. **Герб Тюмени** — в углу карты (пока версия из открытых источников; заменим на файл от Юлии, когда придёт).

## Реализуемость остального

| Функция | Оценка | Комментарий |
|---|---|---|
| Онлайн-пожертвования | Реализуемо | Нужны реквизиты/механизм от заказчика. Варианты: ссылка на платёжную страницу фонда (просто), приём через ЮKassa/СБП (нужно юрлицо-получатель и договор с провайдером). UI уже готов. |
| Рейтинг от посетителей | Реализуемо с оговоркой | Без регистрации возможна только анонимная оценка (защита: ограничение по IP/куке — от накруток полностью не спасает). Альтернатива: рейтинг ведёт редакция. Сейчас поле заполняется в админке. |
| Отзывы | Ключевая развилка | Без регистрации нет авторства. Варианты: (а) без отзывов; (б) анонимные отзывы с премодерацией в админке — реализуемо за 1–2 дня; (в) вход через VK ID/Госуслуги — заметно дороже. Рекомендуем (б), если отзывы нужны. |
| Автосбор мероприятий из внешних источников | Отложено обоснованно | Данные разрозненны, надёжного маппинга «событие → памятник» нет. Ручной ввод уже работает и покрывает потребность. |
| Тур-маршруты, ИИ-гид | Отложено | Технически реализуемо на текущем стеке (строить маршрут по точкам категории/района), но это отдельная итерация. |
| 3D-модели | Уже поддерживаются | Загрузка .glb в админке есть с прошлых итераций; решено не наполнять. |

## Ждём от заказчика

- Файл герба и лекала (Юлия) — заменим временный герб.
- Решение по рейтингу и отзывам (см. таблицу).
- Реквизиты/механизм пожертвований.
- Структуру секций описания (поле уже гибкое — примем любую).
- Решение по источнику фотографий (Яндекс / заказчик).
